## v4施策一覧

### 施策A2: レースドラマ描写（展開ナラティブ）
**目的**: 3連単的中レースに「なぜ当たったか」のストーリーを追加
- 結果データから決まり手・着順を使い、展開を自動生成
- 例: 「3号艇がまくり差しで2号艇を捉え、鮮やかに先頭へ」
- 的中レースのハイライトに組み込む（最大3レース分）
- **対象記事**: accuracy（結果記事）

### 施策B2: グレード別 視覚マーカー
**目的**: 予測信頼度を一目で判別可能に
- S → ◎（二重丸）, A → ○, B → △, C → ✕
- grades記事の各レースにマーカーを付与
- **対象記事**: grades（予測グレード記事）

### 施策D1: タイトル感情フック強化
**目的**: note.com内検索＋タイムラインでのCTR向上
- 結果記事: 「万舟的中！」「3連単5本的中」「ROI 150%」等を動的にタイトルに
- 予測記事: 「本日のSランク〇場」をタイトルに
- 現行の日付のみタイトルからの脱却
- **対象**: 全記事タイプのタイトル生成ロジック

### 施策D3: ハッシュタグ最適化
**目的**: note.com内・Google検索での発見性向上
- 固定タグ: #競艇予想 #ボートレース #AI予測 #舟券
- 動的タグ: 場名（#住之江 #平和島）、高配当時（#万舟）
- 既存のタグ生成ロジックを強化
- **対象**: 全記事タイプ

### 施策C4: メンバーシップ訴求に実績数値
**目的**: 抽象的→具体的に転換し、加入率向上
- 「先月の3連単〇本的中が全部読めます」
- 「累計〇レース分析の全データにアクセス」
- membership記事に過去30日の実績を自動挿入
- **対象記事**: membership

### 施策A4: 翌日プレビュー強化
**目的**: 結果記事→翌日の予測記事への導線
- 結果記事の末尾に「明日の注目」セクション
- 注目場の理由: SG/G1開催、好成績場、荒れやすい場
- **対象記事**: accuracy（結果記事）
